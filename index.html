BEDES compliant?

BEDESbot is a simple way to to check if your spreadsheet headers are BEDES compliant. Paste in a column header to check it.

Using BEDESbot

Paste or type your column header into the box below.

 

{{ header }}

type: {{ columns[header] }}

units: {{ units[header] }}
